# Data Analysis Practice for AI Annotation Projects

This repository contains sample Excel files and practice datasets for managing and analyzing image annotation tasks.

## Contents

- `task_tracking.xlsx` – Sample file to track annotation progress, accuracy, and review notes.
- `accuracy_reports.xlsx` – Sample analysis of completed annotations and error rates.
- `sample_dataset/` – Small dataset of images to practice labeling and data management.

## Purpose

The repository demonstrates:

- Data tracking and organization skills for remote annotation projects.
- Ability to measure accuracy and quality in AI training datasets.
- Basic data analysis skills using Excel.

## Tools Used

- Microsoft Excel
- Google Sheets
- CVAT / Labelbox (for annotations)

## Notes

- These files are examples for showcasing task management and data quality assurance skills.
- Ideal for remote AI/data annotation roles.
